<Blurb>To remove the default blue underline from an anchor tag add: </Blurb>

```html
text-decoration: 'none'
```
<Blurb>Example usage with inline styling</Blurb>

```html react-live

<a href="somewhere-cool" style={{textDecoration: 'none'}}>I have no underline!</a>

```
