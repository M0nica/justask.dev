<Text>Two ways to flattern a nested array. </Text>
<Text>The first is the more traditional approach using reduce and concat </Text>

```js
const myArray = [['1', '2', '3'], ['4', '5']];

const flatArray = myArray.reduce((total, amount) => total.concat(amount));

```

<Text>The second uses the new flat() method - <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/flat">https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/flat</a> </Text>
<Text>As an argument you can pass in the depth </Text>

```js
const myFlatArray = [['1', '2', '3'], ['4', '5']].flat();

```
